🎯 最大エントロピー原理

原理

制約条件の下で、エントロピーを最大化する分布を選ぶ

意味

「知っていること以外は何も仮定しない」

例

  • 制約なし → 一様分布
  • 平均値が既知 → 指数分布
  • 平均と分散が既知 → 正規分布

発散段階との対応

最大エントロピー原理 = 「偏見を持たない」 = 批判禁止

🌐 発散段階の数理化（2）- 多様体上での探索

状態空間

多様体M（確率分布の空間）

探索戦略

ランダムウォーク

数式

dθ/dt = √(2/β) η(t)
  • θ = パラメータ
  • η(t) = ガウスノイズ
  • β = 逆温度（小さいほど探索範囲が広い）

βは「探索の広さ」を制御するパラメータ

🌡️ βパラメータの意味

物理学のアナロジー

統計力学の温度 T と β = 1/T の関係

β → 0（高温）

  • ランダムウォークが激しい
  • 広範囲を探索
  • 発散段階

🔥

β → ∞（低温）

  • ほとんど動かない
  • 局所的な探索
  • 収束段階（次回）

❄️

重要ポイント

βは創造過程の「段階切り替えスイッチ」
